# Password Generator Web App

## Features

- User can enter password length and generate a secure password.  
- Password strength indicator: Weak / Medium / Strong.  
- Show/Hide button to toggle password visibility.  
- Copy to Clipboard button for easy use.  
- Attractive design with gradient backgrounds, hover effects, and shadows.

## Description

This web application allows users to quickly generate secure passwords according to their preferred length. It provides real-time feedback on password strength, ensuring users create strong and safe passwords. The interface is designed to be visually appealing with gradient backgrounds, subtle shadows, and interactive hover effects.

## How to Use

1. Enter the desired password length.  
2. Click the "Generate" button to create a password.  
3. Check the password strength indicator.  
4. Use the Show/Hide button to toggle visibility.  
5. Click "Copy to Clipboard" to copy the password for use.
